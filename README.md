# 💳 LoanAssist EMI Default Prediction Platform

A **Machine Learning based Flask web application** that predicts whether a customer is likely to repay a loan or default. The platform provides **real-time risk prediction, analytics dashboards, graphical insights, PDF reports, and customer management** to support smarter loan decision-making.

---

## 📌 Overview

LoanAssist EMI Default Prediction Platform is developed using **Python, Flask, and Scikit-Learn**. It predicts loan repayment behavior using financial and credit-related information.

The application provides:

- 🤖 Real-time EMI Default Prediction
- 📊 Interactive Analytics Dashboard
- 📈 Model Performance Metrics
- 📄 PDF Report Generation
- 📁 CSV Export
- 📉 Graphical Analysis
- 📝 Prediction History

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/virajkotawadekar/LoanAssist-EMI-Default-Prediction.git
```

Move into project directory

```bash
cd LoanAssist-EMI-Default-Prediction
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser

```text
http://127.0.0.1:5000
```
